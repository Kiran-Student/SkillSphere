SkillSphere
SkillSphere is my **first web development project**, created to practice and apply the fundamentals of **HTML, CSS, and JavaScript.
This project represents my starting point in web development and will be **continuously improved** as I learn new technologies.

Features
- Clean and simple user interface
- Responsive layout (basic)
- JavaScript interaction
- Beginner-friendly structure
  
Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

Purpose of This Project

- To build confidence using GitHub
  
Future Improvements

- More interactivity
- Mobile-first design
- Backend integration (later)
